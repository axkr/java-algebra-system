Download

  • jas.jar execute with java -jar jas.jar RunGB <args>

    Execution of RunGB or other classes with main() method. Contains class and java files.

  • JAS.jar execute with java -jar JAS.jar <RungGB args>

    Execution of RunGB. Contains class and java files.

  • Required libraries: lib/log4j.jar (from Apache Log4j), lib/junit.jar (from Junit), lib/tnj.jar (for Thread and Network Programming).
    In case you have the first two ones, just ensure they are in the correct location (./lib/). The last one you most likely do not have and you must download it.

    Installation: Download the libraries and put them in a directory lib. Create an empty directory test for the output of log4j.

Jython Interpreter

jas.py (imported by other Python files).

To install jas with jython you have to add the above libraries (log4j.jar, junit.jar, tnj.jar and JAS.jar) to the Java classpath.
