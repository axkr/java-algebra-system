Generic coefficients power series package.

This package contains classes for univariate and multivariate power series arithmetic in classes UnivPowerSeries, UnivPowerSeriesRing MultiVarPowerSeries and MultiVarPowerSeriesRing over coefficient rings which implement the RingElem interface.
