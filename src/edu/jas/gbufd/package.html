Groebner bases using unique factorization classes package.

This package contains classes for polynomial pseudo reduction and Groebner bases using pseudo reduction GroebnerBasePseudoSeq and GroebnerBasePseudoRecSeq. Groebner bases for polynomial rings over regular rings with pseudo reduction (direct products of fields or integral domains) are implemented in RGroebnerBasePseudoSeq.
